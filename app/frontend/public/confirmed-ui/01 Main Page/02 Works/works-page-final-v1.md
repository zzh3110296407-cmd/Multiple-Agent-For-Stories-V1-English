# Works Page v4

日期：2026-06-19

## 本版调整

用户反馈：返回按键更适合放在左下角。

v4 在 v3 的基础上只做一个结构调整：

- 移除顶部左侧返回按钮。
- 在页面左下角增加返回按钮。
- 保留作品集三栏结构、卡片样式、右侧当前选中预览、顶部搜索与排序工具。

## 设计判断

返回按钮属于页面级退出动作，不是作品集筛选、排序或创建故事的一部分。放在左下角后，顶部区域可以更专注于作品集操作，用户视线也不会在标题、搜索栏和返回之间产生竞争。

左下角位置与页面留白结合较好，不压住筛选栏和作品卡片，也方便后续做丝滑页面返回动画，例如页面向右下方收束、背景飞龙纹理轻微回退到主页。
