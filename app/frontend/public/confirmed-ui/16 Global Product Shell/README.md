# 16 Global Product Shell

Status: accepted on 2026-07-05.

This folder records global UI states that do not belong to a single creation module.

## Pages

- `01 Workspace Unavailable State` - product shell remains visible while the current workspace is blocked, unavailable, or waiting for a safe redirect.
- `02 Normal Shell Navigation And Status` - normal global shell state with status bar, workspace navigation, selected/locked/hidden states, mode switch, progress header, and responsive behavior.
