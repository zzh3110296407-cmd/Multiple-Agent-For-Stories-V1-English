# 07 导入故事 / 故事分析器最终确认版

当前确认流程：

1. `01 Import Source`：导入故事源、上传文件或接收分析器产物。
2. `02 Analyzing`：展示 Analyze Stories 分析、校验、报告提取和候选预检过程。
3. `03 Analysis Result Overview`：审阅分析结果，并生成 Framework 候选。
4. `04 Framework Candidate Selection`：选择候选并进入导入编辑会话。

后续连接：

- `04 Framework Candidate Selection` 的可用候选会进入 `06 Framework Composition / 03 Imported Edit Session`。

归档说明：

- `00 Legacy Analyze Stories Overview` 为旧版总览稿，只作参考，不作为当前 UI 接入依据。
- 当前实现交接应以 01-04 四个文件夹中的 HTML、PNG 和 Markdown 为准。
