## 四、大 Framework 节点规律

**词库中定义的宏观节点（基础节点）**

| 节点 | 定义 |
|------|------|
| 开端 | 建立故事世界、核心角色、初始基调与首要问题。让读者理解「这个故事的世界是什么样的」。… |
| 触发事件 | 打破初始平衡的关键事件，让故事从静止进入运动。主角的目标或冲突从此刻真正启动。… |
| 发展 | 主角追逐目标、冲突持续扩大、新人物或复杂性被引入的阶段。故事世界因行动而变化。… |
| 升级 | 代价、危险、误解或压力显著增大。主角面临的困境比之前更严峻，退路收窄。… |
| 反转 | 局势的意义发生根本性变化——不只是结果改变，而是读者对整个局面的理解被重构。… |
| 危机 | 主角必须做出关键选择的时刻。这个选择将决定后续走向，且往往无法两全。… |
| 高潮 | 当前叙事线索或全局冲突达到最高强度的节点。之后叙事开始收势，张力不可能再升高。… |
| 回落 | 高潮后的结果逐步展开。后果、损失与变化浮现，世界开始重新找到平衡。… |
| 解决 | 主要冲突被解决、转化或放弃。不一定是happy ending，但核心张力得到了某种形式的释放。… |
| 余波 | 展示新世界状态、情感落点与后续可能性。人物和世界在经历后是什么样的。… |
